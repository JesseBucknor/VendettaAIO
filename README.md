# VendettaAIO

> **Archived Software Engineering Portfolio Project**

VendettaAIO was a commercial software project focused on automating online retail purchasing workflows through a configurable task management platform. The project provided hands-on experience leading the software development lifecycle—from product planning and technical requirements to developer coordination, testing, and iterative feature refinement.

Although the project is no longer under active development, it remains an important part of my software engineering portfolio because it demonstrates product leadership, software architecture planning, and real-world development experience.

VendettaAIO was designed to streamline online purchasing workflows through configurable automation and a modular software architecture.
